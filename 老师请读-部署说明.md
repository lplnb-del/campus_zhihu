## 🚀 快速启动

### 第一步：安装依赖

**后端依赖**（Maven自动下载）：
```bash
cd backend
mvn clean install
```

**前端依赖**（需要先安装Node.js 18+）：
```bash
cd frontend
npm install
```

> **说明**：
> - 后端依赖约80MB，首次运行需要下载时间
> - 前端依赖约216MB，首次运行需要下载时间
> - 建议在联网环境下执行安装

### 第二步：启动数据库

**启动MySQL 8.0**：
```bash
# 创建数据库
mysql -u root -p
CREATE DATABASE campus_zhihu CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

**导入初始化脚本**：
```bash
mysql -u root -p campus_zhihu < database/init.sql
```

**导入测试数据**（可选）：
```bash
mysql -u root -p campus_zhihu < database/test_data_final.sql
```

### 第三步：配置后端

编辑 `backend/src/main/resources/application.yml`：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/campus_zhihu
    username: root
    password: 你的MySQL密码
  data:
    redis:
      host: localhost
      port: 6379
```

### 第四步：启动后端

```bash
cd backend
mvn spring-boot:run
```

后端服务启动在 `http://localhost:8080`

### 第五步：启动前端

```bash
cd frontend
npm run dev
```

前端服务启动在 `http://localhost:5173`

## ⚠️ 注意事项

1. **端口占用**：确保8080（后端）和5173（前端）端口未被占用
2. **数据库编码**：MySQL数据库必须使用utf8mb4编码
3. **JDK版本**：需要JDK 17或更高版本
4. **Node版本**：需要Node.js 18或更高版本
5. **MinIO服务**：图片上传功能需要MinIO服务（可选）
